PROYECTO

Proyecto propuesto como integrador final del curso de desarrollo Full-Stack. Se utilizo Spring como Framework, HTML, CSS y Bootstrap para el Front, MySQL como base de datos. Entre sus funciones se destacan, el acceso mediante LOGIN, registro, modificación, eliminacion y consulta de datos (CRUD). Proyecto en constante modificación.
